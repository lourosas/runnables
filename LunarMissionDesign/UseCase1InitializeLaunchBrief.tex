\textbf{Initialize Launch}
\begin{enumerate}
   \item Flight Controller Enters The Mission Parameters File
   \item System Opens the Parameters File
   \item System Reads the Parameters File Data
   \item The Launching Compoents feed the System its data in real time
   \item System alerts the flight director of the intialization of the
      mission paramters
   \item System Compares the input data to the parameter file data
   \item System determines if there are any errors
   \item System informs the Flight Controller of the current data and
      error state
\end{enumerate}
