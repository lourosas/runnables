\textbf{Initialize Fuel Pipe--Brief Format}
\begin{enumerate}
   \item Flight Director Enters the Mission Parameters File (See 
      \textit{Use Case 1: Initialize Launch})
   \item System finds the Pipe Parameter File name and location
   \item System goes to the Pipe Parameter File location
   \item System Opens the Pipe Parameter File
   \item System reads the Pipe Parameter File Data
   \item System initializes the Pipe Data with the appropriate Pipe 
      Data read in from the Pipe Parameter File
\end{enumerate}
