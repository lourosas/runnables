\textbf{Request Mission Data--Brief Format}
\begin{enumerate}
\item With the (Lunar Mission) System running, querry the desired
Data
\item Data Feeder returns the requested data
\item Depending on the Current State, System Pauses and then repeats,
by returning to number 1
\item Continues for the duration the requested data is available
(entire mission, phase or state)
\end{enumerate}
