\textbf{Initialize Fuel Tank--Brief Format}
\begin{enumerate}
   \item Flight Director Enters the Mission Parameters File (See 
      \textit{Use Case 1: Initialize Launch})
   \item System finds the Tank Parameter File name and location
   \item System goes to the Tank Parameter File location
   \item System Opens the Tank Parameter File
   \item System reads the Tank Parameter File Data
   \item System initializes the Tank Data with the appropriate Tank 
      Data
\end{enumerate}
