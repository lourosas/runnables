\textbf{Enter State Substate--Brief Format}
\begin{enumerate}
   \item With the System Running, the System will periodically change
      State and Substate
   \item System indicates the changes to the Data Feeder
   \item Data Feeder determines if the System's State change is
      relevant to its activities
   \item Data Feeder changes states if Relevant
\end{enumerate}
