\textbf{Initialize Engine--Brief Format}
\begin{enumerate}
   \item Flight Director enters the Mission Parameters File (See 
      \textit{Use Case 1:  Initialize Launch}) 
   \item System find the Engine Parameters File name and location
   \item System goes to the Engine Parameters File location
   \item System opens the Engine Parameters File
   \item System reads the Engine Parameters File Data
   \item For each engine, System initializes the Engine Data with the
      appropriate Engine Data read in from the Engine Parameters File
\end{enumerate}
