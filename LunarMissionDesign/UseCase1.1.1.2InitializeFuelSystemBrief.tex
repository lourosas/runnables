\textbf{Initialize Fuel System--Brief Format}
\begin{enumerate}
   \item Flight Director Enters the Mission Parameters File (See 
      \textit{Use Case 1: Initialize Launch})
   \item System Initializes each Fuel Tank (See 
      \textit{Use Case 1.1.1.2.1:  Initialize Fuel Tank})
   \item System Initializes each Fuel Pump (See 
      \textit{Use Case 1.1.1.2.2:  Initialize Fuel Pump})
   \item System Initializes each Fuel Pipe (See 
      \textit{Use Case 1.1.1.2.3:  Initialize Fuel Pipe})
   \item System queries \textbf{or} receives Initialization Status
      from each Fuel Tank (See 
      \textit{Use Case 1.1.1.2.1:  Initialize Fuel Tank})
   \item System queries \textbf{or} receives Initialization State from
      each Fuel Pump (See 
      \textit{Use Case 1.1.1.2.2:  Initialize Fuel Pump}) 
   \item System queries \textbf{or} receives Initialization State from
      each Fuel Pipe (See 
      \textit{Use Case 1.1.1.2.3:  Initialize Fuel Pipe})
   \item System returns Initialization Data to the Flight Director
\end{enumerate}
