\textbf{Request to load Parameter File--Brief Format}
\begin{enumerate}
   \item With the Data Feeder running, the (Lunar Mission) System/
      System Administrator Requests the Data Feeder load a
      Parameter File
   \item The System/System Administrator sends the name and location
      of the Parameter File to the Data Feeder
   \item The Data Feeder Opens the Parameter File
   \item The Data Feeder loads all the Mission Data Parameters into
      the Feeder
   \item The Data Feeder periodically generates the Mission Data based
      on the Parameter File requirements
\end{enumerate}
