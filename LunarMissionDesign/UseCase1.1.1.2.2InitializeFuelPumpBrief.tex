\textbf{Initialize Fuel Pump--Brief Format}
\begin{enumerate}
   \item Flight Director Enters the Mission Parameters File (See 
      \textit{Use Case 1: Initialize Launch})
   \item System finds the Pump Parameter File name and location
   \item System goes to the Pump Parameter File location
   \item System Opens the Pump Parameter File
   \item System reads the Pump Parameter File Data
   \item System initializes the Pump Data with the appropriate Pump 
      Data read in from the Pump Parameter File
\end{enumerate}
